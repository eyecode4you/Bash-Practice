## Interacting with the GRUB2 Bootloader Menu
Boot or reboot system to get to GRUB2 bootloader menu
- e to edit
- Ctrl+c to enter GRUB2 cmd line, Esc to exit - doesn't save changes made
- Ctrl+x to save and boot
